Universe in Pixels

Java Android Hackathon

Universe in Pixels is an Android application that was developed at the CU Hackathon 2021. The app lets users view all the photos that were published on NASA's Astronomy Picture of the Day, view all the pictures taken by the Spirit, Opportunity, and Curiosity Rovers on Mars, and track and view the real-time location of the International Space Station on Google maps. This app implements NASA's APIs for the image data, and wheretheiss.at 's API for ISS tracking.
